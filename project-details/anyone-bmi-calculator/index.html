I came up with this idea when I was working on my company project, and the designer came and talked to me about the possibility to make some graphics change based on user's age, gender and BMI. Although the idea was banned due to resources constraints, I transform the technical solution to my personal projects, just for fun.

"任何仁", which his name has the meaning of "Anyone", is a mascot wearing blue all-inclusive tights launched by Hong Kong Fire Service Department. I am using this mascot to represent anybody who are using this BMI calculator!

This simple progressive web application was built using Vue.js.
