"蘋果廣告製作公司" (Lucky Advertising Company), is famous for putting a TV commercials with video of fireworks as background, and a big yellow text in the middle showing the company name. As its' style and quality looks so out of step with time, the broadcast has caused a lot of netizens to discuss.

I build this simple progressive web application with HTML canvas and JavaScript (ES6), which combines a GIF file of fireworks with text input by user.
