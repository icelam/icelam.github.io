Progressive Web App demo which fetch Hong Kong hottest news from News API. Written in ReactJS, Material-UI, with help of Create React App.

As projects initialized using Create React App already have default configuration of service worker through Workbox plugin, I need to find a way to customizing it without ejecting. I end up writing my own service worker template.
