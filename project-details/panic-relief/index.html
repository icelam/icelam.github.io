I have been suffering from anxiety and hyperventilation recently, so I have created this pure CSS animation to help me relax and relief the overbreathing syndrome.

The animation illustrates a breathing technique called "Square Breathing", which includes the following repeated steps, each for 4 seconds:

  1. Inhale
  2. Hold
  3. Exhale
  4. Hold
I hope this animation can help anyone who is becoming anxious having a panic attack.
