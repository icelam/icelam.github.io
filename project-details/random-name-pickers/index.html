This website was originally developed for use in my company's lucky draw day to celebrate Chinese New Year. Given that I can control the browsers and OS used for running the lucky application, I decided to use some relatively new Web APIs to make the project more fun.

The spinning animation used in slot was accomplished by Web Animations API, which is a perfect solution when want your animation is sensitive to JavaScript states. Although Web Animations API is still in working draft, it is having a good support on legacy browsers like IE 11 and iOS 12 with polyfill applied. My experience on this new API is quite good despite issues on Safari had annoyed me for couples of hours.

Web Audio API is also used to make the application more inactive. Without any knowledge on multimedia or music theory, I managed to make some simple sound effects by mapping sound frequency to piano keys. The process is quite interesting although I spent quite a couple of hours coding sound effects that only last for few seconds.
